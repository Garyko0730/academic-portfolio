## 项目概述

展示 OpenClaw 多 Agent 状态的像素办公室网页 UI，实时可视化 AI Agent 的工作状态和进度。

## 技术栈

- **前端**: Phaser 3
- **后端**: Flask (Python)
- **通信**: REST API + WebSocket

## 特色功能

- 像素风格办公室场景
- 多 Agent 状态实时同步
- Agent 任务进度可视化
